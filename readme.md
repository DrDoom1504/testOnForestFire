# Linear Regression Project

## Overview
This project implements linear regression models for predicting forest fire occurrences using the Algerian Forest Fires dataset. It includes data preprocessing, model training, and a web application for predictions.

## Features
- Data cleaning and preprocessing
- Multiple regression models (Linear, Ridge, Polynomial)
- Web interface for model predictions
- Jupyter notebooks for experimentation

## Dataset
The project uses the Algerian Forest Fires dataset, which contains meteorological data and fire occurrences from two regions in Algeria (Bejaia and Sidi Bel Abbes).

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python application.py`

## Usage
- Train models using the Jupyter notebooks
- Access the web app at `http://localhost:5000` (assuming Flask app)

## Project Structure
- `application.py`: Main Flask application
- `models/`: Trained model files
- `templates/`: HTML templates for the web interface
- Notebooks: Model training and analysis scripts
